# DevSpaces 🚀

DevSpaces is an open-source web application that allows developers to create, manage, and access their GitHub Codespaces from one centralized dashboard.

## Features

* View all GitHub Codespaces
* Create new Codespaces
* Start and stop Codespaces
* Delete Codespaces
* Monitor Codespace status
* Access Codespaces from one dashboard
* Integrated web terminal *(planned)*
* GitHub OAuth authentication *(planned)*
* Multi-account support *(planned)*

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* GitHub Codespaces API
